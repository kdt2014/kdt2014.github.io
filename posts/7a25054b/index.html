Ubuntu安装指定版本的Nodejs

  • 更新软件源

      sudo apt update

  • 在 Ubuntu安装 Node.js 14

      curl -sL https://deb.nodesource.com/setup_14.x | sudo bash -

    想安装15版本的就把上面的14改成15

  • 安装nodejs

      sudo apt -y install nodejs

  • 验证安装的 Node.js 版本

      node  -v
